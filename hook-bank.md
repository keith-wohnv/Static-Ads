## Goal Filter Key

Use these tags to find hooks for your specific situation:

| Tag | Goal | Description |
|-----|------|-------------|
| `ctr-low` | CTR is low | People scroll past |
| `hook-weak` | Weak hook | Drop-off in 0-3 seconds |
| `cvr-low` | CvR is low | Clicks but no sales |
| `tof` | Need TOF hooks | Cold audience, first touch |
| `bof` | Need BOF hooks | Retargeting, close the sale |
| `trust` | Build trust | They don't believe you yet |
| `organic` | Organic feel | Needs to feel native |
| `retention` | Low retention | They leave too early |
| `engagement` | Need engagement | Likes, comments, shares |
| `urgency` | Create urgency | Push them to act now |
